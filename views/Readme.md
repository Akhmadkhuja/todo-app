# TO DO APPLICATION

App allows users to track the to dos.

#### Introduction
1.Download the source code

```bash
git clone https://github.com/Akhmadkhuja/todo-app.git
```
or just download the .zip file

2.Install file
```bash
npm install
```

3.Running the app
```bash
node app
```
